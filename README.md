# vim-jss

Vim syntax highlighting for [JSS](https://github.com/JamalMulla/Simulator) code.

## Installation

### Neovim
```
cp jss.vim ~/.config/nvim/syntax/
```


### Vim
```
cp jss.vim ~/.vim/syntax/
```
